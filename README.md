# Proyecto de Data Analytics sobre Accidentes Aéreos

Este es un proyecto de Data Analytics en base al Dataset proporcionado por la Organización de Aviación Civil Internacional (OACI) con el objetivo de analizar los accidentes aéreos en todo el mundo. El propósito principal es recopilar, analizar y visualizar datos relevantes sobre accidentes aéreos para identificar patrones y tendencias en la seguridad de la aviación civil.

## Resumen del Dastaset original

A continuacion se detallan las variables presentes en el Dataset original "AccidentesAviones.csv":

* Unnamed: 0: Columna sin nombre que parece ser un índice
* fecha: Fecha del accidente.
* HORA declarada: Hora declarada del accidente.
* Ruta: Ruta del vuelo.
* OperadOR: Compañía aérea.
* flight_no: Número de vuelo.
* route: Propocito del vuelo.
* ac_type: Tipo de aeronave.
* registration: Registro de la aeronave.
* cn_ln: Número de construcción o de serie / número de línea o de fuselaje.
* all_aboard: Total de personas a bordo.
* PASAJEROS A BORDO: Número de pasajeros a bordo.
* crew_aboard: Número de tripulantes a bordo.
* cantidad de fallecidos: Cantidad de personas fallecidas en el accidente.
* passenger_fatalities: Número de pasajeros fallecidos.
* crew_fatalities: Número de tripulantes fallecidos.
* ground: Número de personas fallecidas en tierra.
* summary: Resumen del accidente aéreo.

### El proyecto se desarrolló en las siguientes etapas:

1. Análisis exploratorio de los datos (EDA): se analizaron los datos de accidentes aéreos, incluyendo estadísticas descriptivas, análisis univariados
2. Creación del Dashboard Interactivo: Se desarrolló un dashboard funcional con el objetivo de presentar los hallazgos y permitir explorar los datos de manera interactiva. El dashboard se encuentra basado en el análisis realizado en la etapa anterior y cuenta con visualizaciones claras y coherentes.
3. Definición de KPIs: Se sugirieron KPI's enfocados reforzar la seguridad en puntos de observacion como son los tipos de aeronave, la operadora y la ciudad de origen.

#### Comentarios finales

Este proyecto de Data Analytics sobre accidentes aéreos ha sido una interesante oportunidad para analizar y comprender la seguridad de la aviación a través de datos históricos. En terminos generales, este proyecto demuestra la importancia de la aplicación de técnicas de Data Analytics en la industria y como el análisis sistemático de datos históricos permite identificar áreas críticas y generar acciones para prevenir futuros accidentes y mejorar la seguridad a nivel mundial.
